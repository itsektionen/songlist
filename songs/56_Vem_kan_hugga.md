Vem kan hugga sig själv i knät,
vem kan slå sig på tummen,
vem kan skära i vännen sin,
utan att dra på munnen?

Jag kan hugga mig själv i knät,
jag kan slå mig på tummen.
Men ej skära i vännen min
utan att dra på munnen.
